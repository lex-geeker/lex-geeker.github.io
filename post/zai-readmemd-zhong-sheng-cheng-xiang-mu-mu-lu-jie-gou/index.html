在Readme.md中生成项目目录结构

2019-10-30

使用Homebrew 安装tree

brew install tree

使用tree在控制台生成当前目录树

// 因为实例项目为 js 项目，所以忽略了 node_modules，可根据真实项目选择性忽略文件目录，防止文件树过长
tree -I node_modules > tree.text

复制目录树文本到readme.md中
